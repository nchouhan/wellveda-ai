# Wellveda AI - Ayurvedic Wellness Assistant

An AI-powered Ayurvedic wellness platform that provides personalized health insights through advanced natural language processing and interactive digital experiences.

## Features

- Personalized Ayurvedic consultations using OpenAI technology
- Support for both English and Hindi languages
- Detailed responses on dosha types, diet recommendations, and wellness practices
- Responsive, modern chat interface with intuitive design
- FAQ section with common Ayurvedic questions
- SEO-optimized for better discoverability

## Technology Stack

- OpenAI Assistants API for intelligent conversations
- TypeScript for type-safe code
- React for the frontend UI
- Node.js with Express for the backend API
- Tailwind CSS for responsive styling
- Vite for fast development and optimized builds

## Getting Started

### Prerequisites

- Node.js 18 or higher
- OpenAI API key

### Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/wellveda-ai.git
cd wellveda-ai
```

2. Install dependencies:
```bash
npm install
```

3. Create a `.env` file in the root directory and add your OpenAI API key:
```
OPENAI_API_KEY=your_api_key_here
```

4. Start the development server:
```bash
npm run dev
```
